Question Generator Grammar

This is the grammar that is used when generating survey questions.

If you would like more information on how it works please visit the Wiki

Legend:

Inline Choice ^$ Option A:Option B$
Variable [name:value]
Key #key.modifiers#

Reserved Variables:

typeAnswer Type (default: 'text')
keyKey used to generate answers
countNumber of answer choices (default: random 3-8, -1 returns all options)
otherIf multiple choice has "other" option
tagOption tagged for later filtering
